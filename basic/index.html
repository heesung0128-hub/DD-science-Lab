⚙️ Antigravity 시스템 설정

실시간 AI 탐구 설계를 사용하기 위한 설정을 관리합니다. 입력하신 API 키는 브라우저의 로컬 환경(LocalStorage)에만 안전하게 저장되며 외부로 수집되지 않습니다.

* Claude API는 브라우저 직접 호출 시 CORS 보안 에러가 나므로 우회 프록시 URL 주소를 입력해 주어야 작동합니다.

* 선택된 AI 공급자의 API 키를 비워둘 경우, AI 제안 시 모의 체험(Simulated) AI 모드로 자동 폴백하여 고품질의 어울리는 주제들을 설계합니다.

선생님이 제공한 공유 링크를 사용하지 않고 개별적으로 실시간 연동을 연결하려면 입력하세요. (비워둘 시 오프라인 모드로 자동 작동)

🤖 제미나이 웹 브릿지 (무료/API 키 없음)

학생의 개인 교육용 제미나이(gemini.google.com) 계정을 활용하는 안전한 연결 방식입니다.

  1. 아래 프롬프트 창 우측의 '복사하고 제미나이로 이동' 버튼을 클릭합니다.
  2. 열린 제미나이 웹페이지 채팅창에 붙여넣고(Ctrl+V) 전송합니다.
  3. 제미나이의 답변 전체를 복사(Ctrl+A 후 Ctrl+C 또는 복사 아이콘 클릭)합니다.
  4. 아래 '제미나이 답변 붙여넣기' 영역에 입력하고 '분석 및 적용하기'를 누르세요.
